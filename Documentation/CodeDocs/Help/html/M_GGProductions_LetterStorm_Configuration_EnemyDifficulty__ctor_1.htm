gameSpeed

The speed at which the screen scrolls and enemies move. The default value is 1.0
initialHealth
Type: SystemInt32
The amount of health the player starts the first level with
damageToPlayerPerHit
Type: SystemInt32
The amount of damage taken by the player when he is hit by a projectile or enemy
maxEnemiesonScreen
Type: SystemInt32
The (approximate) maximum number of enemies displayed on the screen simultaneously
bossHealthHandicap
Type: SystemSingle
The amount by which the boss's health should be decreased or increased. This applies only to parts of the boss that can be destroyed using traditional projectiles (vs letters). The default value is 1.0
enableBossRageMode
Type: SystemBoolean
Whether the boss should charge the user when he fires the wrong letter
bossCannonTrackingAccuracy
Type: SystemSingle
How tightly the boss's cannons lock on to the main character. A value of 1.0f means they are always locked on, while lower values will introduce lag.
bossCannonProjectilesPerSecond
Type: SystemSingle
The number of projectiles per second the boss can fire from one of cannons
bossRageChargeSpeed
Type: SystemSingle
When the boss enters rage mode, the speed at which he will charge the user. A value of 1.0f represents his normal speed.
bigBossWindowToAcceptLetters
Type: SystemInt32
The number of seconds the big boss can accept valid letters. This can be used to determine, for example, how long the boss keeps his mouth open.
bigBossWindowToDeflectLetters
Type: SystemInt32
The number of seconds the big boss will not accept valid letters. This can be used to determine, for example, how long the boss keeps his mouth closed.
